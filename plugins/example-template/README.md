# Example Plugin Template

This folder is a small reference plugin for developers who want a concrete starting point.

It is not meant to be installed as an official Lumio plugin.

## What it shows

- a typical `plugin.json`
- a public-facing plugin README
- a simple changelog format
- the metadata shape expected by the marketplace

## Recommended workflow

If you want to build your own plugin:

1. copy this folder
2. rename the folder
3. change the plugin ID and slug
4. update the README and changelog
5. add a matching entry to `marketplace.json`

For SDK behavior and repository conventions, see:

- `docs/sdk.md`
- `docs/PLUGIN_TEMPLATE.md`
- `CONTRIBUTING.md`
