# YouTube Plugin Privacy Policy

This page describes how the YouTube plugin for Lumio Media Player handles data.

## What the plugin accesses

When a user connects YouTube, the plugin may access account data needed to power
the in-app experience, such as:

- subscribed channels
- playlists
- Watch later
- channel and video metadata

The plugin only uses this data to provide YouTube-related features inside Lumio.

## How data is used

Data retrieved from Google / YouTube is used to:

- authenticate the user
- load personal YouTube views inside Lumio
- build home rows and content grids
- support optional actions such as channel follow state, playlist browsing and
  other account-linked YouTube features

## Storage

Lumio may store cached YouTube metadata and auth session information locally on
the user’s device in order to improve performance and reduce repeated API calls.

## Sharing

The YouTube plugin does not sell user data.

Any Google / YouTube account data used by the plugin is used only for the plugin
experience inside Lumio, except where disclosure is required by law or needed to
operate the service.

## User controls

Users can:

- disconnect their YouTube account from within Lumio
- clear cached YouTube data from the plugin settings
- optionally use their own Google app credentials where supported

## Contact

For questions about this plugin, contact the Lumio project or plugin maintainer
through the channels listed on the repository page.
